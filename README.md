# Akan Concept(Book Lending Website) Landing Page

Bootstrap-based promotional site for the Akan Concept book. The page highlights pricing tiers, reader reviews, and contact options while featuring responsive navigation and modal/offcanvas components.

## Structure
- `AkanConcept.html` – main page tailored for the book concept.
- `index.html` – additional entry point (currently unused demo content).
- `assets/` – logo and imagery (e.g., Asset 2 as logo/favicon, ebook artwork, device mockups).

## Getting Started
1. Open the project folder in your editor.
2. Launch `AkanConcept.html` with Live Server or any static file server.
3. Ensure you have an internet connection to load Bootstrap and icons via CDN.

## Customization Tips
- Update images inside `assets/` to match your branding.
- Edit pricing cards and reviews in `AkanConcept.html` to reflect real content.
- Extend the CSS section in the `<head>` if you need further styling tweaks.
